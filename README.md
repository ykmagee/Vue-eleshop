# vue-eleshop

## 概述

	1.本项目为前后端分离的一个外卖平台
	2.前端技术栈：Vue+vuex+vue-router+axios+ES6+Webpack
	3.后台技术栈：Node.js+Express+MongoDB

## 项目地址：http://ykit.xyz

## 准备环境 

    安装 node.js  
    可选：MongoDB(用于存放用户信息)

## 运行项目 

#### 运行后台(eleshop-server文件夹)

    1.npm install   
    2.npm start   

#### 运行前端(vue-eleshop文件夹)

    1.npm install   
    2.npm run dev
